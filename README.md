The goal of this project is analyzing data from the U.S. Census Bureau and creating a scatter plot 
between two of the data variables such as Healthcare vs. Poverty. The plot represents each state 
with circle elements.

The project was completed using D3.js library.

Files included within this projects:

* this README.md
* d3Style.css
* style.css
* data.csv
* app.js
* index.html
